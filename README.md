# ROV-Scorpion-2.0

This project presents the ROV Scorpion 2.0 prototype, developed as part of a collaborative task.
The prototype was built by Shah M. Seaman and Abu Sufian Robin, with each contributor focusing on their area of expertise:

Shah M. Seaman: Configured and developed the codebase for the prototype.

Abu Sufian Robin: Managed the dataset, including model training and fine-tuning.

This repository contains the core code and supporting files required to operate the ROV Scorpion 2.0 prototype.
